# Glossary

- **App**: the hosted website experience in `excalidraw-app/`.
- **Library**: the reusable React component shipped as `@excalidraw/excalidraw` in `packages/excalidraw/`.
- **Monorepo**: a single repository that contains multiple related packages and applications.
- **Workspace**: Yarn workspaces used to manage multiple packages in one repo.
- **Snapshot test**: a test that stores expected output (often UI-related) and fails when the output changes.
- **Typecheck**: running TypeScript’s compiler checks to ensure types are consistent across the codebase.
